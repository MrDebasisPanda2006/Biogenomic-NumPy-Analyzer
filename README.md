# Biogenomic NumPy Analyzer (v1.0)

## Project Overview
This is a high-performance bioinformatics tool developed to process and analyze genomic data. Using **NumPy vectorization**, this code can perform its analysis upto **300 million DNA bases** (similar to human chromosome DNA) in just a few seconds.

## Key Features
* **Massive Scale:** Generates synthetic DNA sequences of 3 times 10^8 elements using np.random.
* **Vectorized GC-Content:** Calculates chemical stability using optimized boolean masking.
* **Codon Analysis:** Reshapes 1D data into 2D matrices to identify **Start (ATG)** and **Stop (TAA, TGA, TAG)** codons across 100 million rows simultaneously.
* **High Performance:** Built with zero Python loops to ensure maximum CPU efficiency.

## Technical Skills Demonstrated
* **Library:** NumPy
* **Concepts:** Array Reshaping, Boolean Indexing, Vectorization, Axis-wise Reduction.
* **Standards:** Modular code structure using `if __name__ == "__main__":`.
